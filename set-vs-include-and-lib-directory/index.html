设置VS包含目录和库文件目录

发表于 2015-09-24 | 分类于 Windows |

根据设置结果适用的范围，可分为以下几种情况：

设置为仅用于当前project

选中当前project -> Propertis -> Configration Properties，
C/C++ -> General：Additional Include Directories，设置为d:\third_party\boost
Linker -> General：Additional Library Directories，设置为d:third_party\stage\lib

设置为仅用于当前Solution

选中当前project -> Propertis -> Configration Properties -> VC++ Directories，
Include Directories，设置为d:\third_party\boost
Library Directories，设置为d:third_party\stage\lib

设置为OS当前用户的VC++环境（即当前用户VC++所创建的所有Solution）

在某个已经打开的工程下，切换到Property Manager选项卡（解决方案选项卡，或者VIEW/Other Windows/Property Manager菜单项） -> 当前属性配置 -> Microsoft.Cpp.Win32.user -> Common Properties -> VC++ Directories，
Include Directories，设置为d:\third_party\boost
Library Directories，设置为d:third_party\stage\lib

设置为OS所有用户的VC++环境

可以通过编辑Microsoft.Cpp.Default.props、Microsoft.Cpp.props来实现。
